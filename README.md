# TEAZO_Mock_UI

## Run locally

This is a static site (`index.html`, `styles.css`, `app.js`), so you can run it in either of these ways:

1. Open `index.html` directly in your browser.
2. Or serve it with a local HTTP server (recommended).

### Option A: Open directly

1. In File Explorer, go to this project folder.
2. Double-click `index.html`.

### Option B: Run with a local server

From the project root, run one of the following:

#### Python 3
```bash
python -m http.server 5500
```

Then open:

`http://localhost:5500`

#### Node.js (`npx serve`)
```bash
npx serve .
```

Then open the URL shown in the terminal (usually `http://localhost:3000`).
